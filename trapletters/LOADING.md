Le jeu s'initialise... Si le dictionnaire prend du temps, patientez quelques secondes.

- Chargement du dictionnaire local `dict/fr.txt`.
- Si le fichier est manquant, le jeu utilisera une liste interne de secours.

Support pour grand dictionnaire français:
- Si vous placez un fichier plus complet `dict/fr-full.txt` (conjugaisons + formes), le jeu l'utilisera en priorité pour le français.
- Formats acceptés: une ligne = un mot en MAJUSCULES (UTF-8). Exemples de sources: Lefff, Lexique, dumps Wiktionary (pré-traités).

Si vous voulez un dictionnaire complet, mettez `fr-full.txt` dans le dossier `dict/`.

Si vous voyez des erreurs JavaScript dans la console, ouvrez la console du navigateur pour diagnostiquer.
